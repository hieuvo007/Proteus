> Detailed analysis and simulation of Bipolar Junction Transistors (BJT) and Metal-Oxide-Semiconductor Field-Effect Transistors (MOSFET) using Proteus Design Suite.

## ðŸ“– Overview
This project focuses on the comparative study and application of **BJT** and **MOSFET** technologies. The simulations demonstrate key characteristics, biasing methods, and switching behaviors of these semiconductor devices.

**Key File:** `Bjt_mosfet.pdsprj`

## ðŸš€ Features
- **BJT Analysis:**
  - NPN/PNP configurations.
  - Current amplification ($h_{FE}$) verification.
  - Saturation and Cut-off regions visualization.
- **MOSFET Analysis:**
  - Gate-Source threshold voltage ($V_{GS(th)}$) testing.
  - High-speed switching capabilities.
  - Comparison of input impedance between BJT and MOSFET.
